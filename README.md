# 🌆 URIS - Urban Resource Intelligence System

## 📌 Project Overview

URIS (Urban Resource Intelligence System) is a smart city project that predicts and optimizes the allocation of essential resources such as **water, electricity, and waste management** using Machine Learning.

---

## 🚀 Features

* 📊 Predicts resource demand (Water, Electricity, Garbage)
* ⚡ Optimized allocation using ML models
* 📈 Interactive dashboard using Streamlit
* 🧠 Multiple ML pipelines for different resources

---

## 🛠️ Tech Stack

* Python
* Machine Learning (Scikit-learn)
* Streamlit (Frontend)
* Pandas, NumPy
* Git & GitHub

## ▶️ How to Run the Project

1. Clone the repository:

```
git clone https://github.com/avinash2004avii-jpg/URIS_Urban-Resource-IntelligenceSystem.git
```

2. Navigate to project folder:

```
cd URIS_Project
```

3. Install dependencies:

```
pip install -r requirements.txt
```

4. Run the dashboard:

```
streamlit run dashboard_app.py
```
